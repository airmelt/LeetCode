# 2180 Count Integers With Even Digit Sum 统计各位数字之和为偶数的整数个数

__Description:__

Given a positive integer `num`, return _the number of positive integers __less than or equal to___ `num` _whose digit sums are __even___.

The digit sum of a positive integer is the sum of all its digits.

__Example:__

Example 1:

```text
Input: num = 4
Output: 2
Explanation:
The only integers less than or equal to 4 whose digit sums are even are 2 and 4.
```

Example 2:

```text
Input: num = 30
Output: 14
Explanation:
The 14 integers less than or equal to 30 whose digit sums are even are
2, 4, 6, 8, 11, 13, 15, 17, 19, 20, 22, 24, 26, and 28.
```

__Constraints:__

- `1 <= num <= 1000`

__题目描述:__

给你一个正整数 `num` ，请你统计并返回 __小于或等于__ `num` 且各位数字之和为 __偶数__ 的正整数的数目。

正整数的 各位数字之和 是其所有位上的对应数字相加的结果。

__示例:__

示例 1：

```text
输入：num = 4
输出：2
解释：
只有 2 和 4 满足小于等于 4 且各位数字之和为偶数。
```

示例 2：

```text
输入：num = 30
输出：14
解释：
只有 14 个整数满足小于等于 30 且各位数字之和为偶数，分别是： 
2、4、6、8、11、13、15、17、19、20、22、24、26 和 28 。
```

__提示：__

- `1 <= num <= 1000`

__思路:__

```text
1. 打表
由于 n 的范围较小, 我们可以直接打表, 从 1 到 1000, 统计每个数的各位数字之和是否为偶数
时间复杂度为 O(1), 空间复杂度为 O(N)
2. 暴力
直接遍历 1 到 n, 统计每个数的各位数字之和是否为偶数
时间复杂度为 O(NlogN), 空间复杂度为 O(1)
3. 数学
[0, 10) 中有 5 个数的各位数字之和为偶数
[0, x], x <= 10 中有 x / 2 + 1 个偶数, (x + 1) / 2 个奇数
记 d = num / 10, r = num % 10, 则 num = 10 * d + r
注意到 [0, 10 ^ k] 中有一半的数的各位数字之和为偶数, 另一半为奇数
所以若 d 的各位数字之和为奇数, 则 [10 * d, 10 * d + x] 中有 x / 2 + 1 个偶数, 否则 有 (x + 1) / 2 个偶数
时间复杂度为 O(logN), 空间复杂度为 O(1)
```

__代码:__

__C++__:

```C++
class Solution 
{
public:
    int countEven(int num) 
    {
        int d = num / 10, r = num % 10, result = d * 5, s = 0;
        while (d)
        {
            s += d % 10;
            d /= 10;
        }
        return result - 1 + ((s & 1) ? ((r + 1) >> 1) : (r >> 1) + 1);
    }
};
```

__Python__:

```Python
class Solution:
    def countEven(self, num: int) -> int:
        return len([i for i in range(1, num + 1) if not sum(int(j) for j in str(i)) & 1])
```
