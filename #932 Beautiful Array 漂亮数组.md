# 932 Beautiful Array 漂亮数组

__Description__:
An array nums of length n is beautiful if:

nums is a permutation of the integers in the range [1, n].
For every 0 <= i < j < n, there is no index k with i < k < j where 2 \* nums[k] == nums[i] + nums[j].
Given the integer n, return any beautiful array nums of length n. There will be at least one valid answer for the given n.

__Example:__

Example 1:

Input: n = 4
Output: [2,1,4,3]

Example 2:

Input: n = 5
Output: [3,1,2,5,4]

__Constraints:__

1 <= n <= 1000

__题目描述__:
对于某些固定的 N，如果数组 A 是整数 1, 2, ..., N 组成的排列，使得：

对于每个 i < j，都不存在 k 满足 i < k < j 使得 A[k] \* 2 = A[i] + A[j]。

那么数组 A 是漂亮数组。

给定 N，返回任意漂亮数组 A（保证存在一个）。

__示例 :__

示例 1：

输入：4
输出：[2,1,4,3]

示例 2：

输入：5
输出：[3,1,2,5,4]

__提示:__

1 <= N <= 1000

__思路__:

分治递归
将所有奇数放在前半部分, 所有偶数放在后半部分
对前半部分和后半部分分别进行递归处理
前半部分按照 2 \* a - 1, 后半部分按照 2 \* a 的方式生成数组
时间复杂度为 O(nlgn), 空间复杂度为 O(nlgn)

__代码__:
__C++__:

```C++
class Solution 
{
public:
    vector<int> beautifulArray(int n) 
    {
        vector<int> result(n, 1);
        helper(result, 0, n - 1);
        return result;
    }
private:
    void helper(vector<int>& arr, int left, int right)
    {
        if (right <= left) return;
        int mid = left + ((right - left) >> 1);
        helper(arr, left, mid);
        helper(arr, mid + 1, right);
        for (int i = left; i <= mid; i++) arr[i] = (arr[i] << 1) - 1;
        for (int i = mid + 1; i <= right; i++) arr[i] <<= 1;
    }
};
```
